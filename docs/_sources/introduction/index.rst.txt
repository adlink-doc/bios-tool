Introduction
=============

What is BMT?
-------------

**BIOS Modification tool** is common utility for all ADLINK COM/SAMRC/Q7 products which allows you to configure the menu of AMI UEFI BIOS binary file. |br|
It only supports WIndows GUI version and all operation are performaed on ROM file only. |br|

.. note::
   It is free utility and 3rd-party license is not required


Features
----------

- **Rename Entries**:     Allow you to renew each node or entries
- **Change Visibility**:  Allow you to hide / show any entries
- **Set Default Values**:  Allow you to configure the values as the default for each entries
- **Change DMI Information**:  Allow you to modify DMI data, such BIOS information, System information or to add OEM strings
- **Track Changes**:  Allow you to track any changes you made.
- **Import setting**:  Allow you import your preference without re-configuration.


Support Operating System
--------------------------

* Windows 7 32bit/64bit
* Windows 10 64bit
